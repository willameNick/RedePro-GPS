# RedeCalc Pro GPS — Modelo Equatorial

Versão ajustada para GitHub Pages.

## Ajustes incluídos

- Poste novo: quadrado vermelho vazado.
- Poste existente: quadrado preto vazado.
- Legenda automática no desenho e no PDF.
- Prancha PDF em A4 retrato no modelo Equatorial.
- Quadro de postes e estruturas acima do carimbo.
- Rede MT em verde e rede BT em preto.
- Transformador e estruturas destacados em magenta.
- Arquivos prontos para publicar no GitHub Pages.

## Como publicar

1. Envie todos os arquivos deste pacote para o repositório.
2. O arquivo principal deve ficar com o nome `index.html`.
3. Vá em **Settings > Pages**.
4. Em **Branch**, selecione `main` e pasta `/root`.
5. Clique em **Save**.

Depois acesse o link do GitHub Pages pelo navegador do celular.
